# components

## Purpose
React components split into two categories: app-specific components for MedAgent features, and generic UI primitives from shadcn/ui.

## Subdirectories

| Directory | Purpose |
|-----------|---------|
| `app/` | MedAgent-specific components — clinician console, approval controls, session UI (see `app/AGENTS.md`) |
| `ui/` | shadcn/ui primitives — button, card, dialog, toast, etc. (see `ui/AGENTS.md`) |

## For AI Agents

### Working In This Directory
- Add new feature components to `app/`, not `ui/`
- `ui/` components are generated by shadcn/ui — modify sparingly
- Import UI primitives from `@/components/ui/...`
